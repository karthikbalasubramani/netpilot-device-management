# NetPilot - O-RAN / IoT Device Management Platform

NetPilot is a cloud-native device management platform built using Go, MongoDB, MQTT, Docker, and Kubernetes.

## Purpose

This project is designed to manage O-RAN/IoT-style devices, collect telemetry, process heartbeat events, generate alerts, and expose secure REST APIs.

## Tech Stack

- Go
- MongoDB
- MQTT
- Docker
- Kubernetes
- Python Device Simulator
- JWT Authentication
